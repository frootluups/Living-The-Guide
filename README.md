# Living-The-Guide
A simple guide I made for anyone who wants to start emulation using Ryujinx (especially for those who want to emulate Tomodachi Life: Living The Dream)

# What you'll need:
- An emulator
- Firmware
- Keys
- Your game of choice (for this one we'll be using Tomodachi Life: LTD)

# Where you'll find them:
- [Ryujinx Canary](https://git.ryujinx.app/ryubing/canary/releases) (Any build should do, I use 1.3.269)
- [Switch Army Knife](https://github.com/dezem/SAK) (To convert NSZ to NSP)
- [Switch Firmware](https://github.com/THZoria/NX_Firmware/releases)
- [Switch Keys](https://prodkeys.net/ryujinx-prod-keys-update/)
- [ROMs of choice](https://fmhy.net/gaming) (This is a compilation of everything free, but simply put I personally use not.ultranx.ru. You need an account to download ROMs but you don't need an email to register)

# What do I do now?
1. Extract your files (except Switch Firmware)
2. Open Ryujinx
